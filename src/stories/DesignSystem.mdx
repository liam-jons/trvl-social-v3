# Glassmorphic Design System

Welcome to the Glassmorphic Design System for TRVL Social v3. This design system provides a trust-centered, modern interface with glassmorphic effects.

## Design Principles

1. **Trust-Centered**: Every element is designed to build user confidence
2. **Accessibility**: WCAG 2.1 compliant with proper color contrast
3. **Responsive**: Mobile-first design with breakpoints at 320px, 768px, 1024px, and 1440px
4. **Performance**: Optimized blur effects for smooth mobile performance
5. **Consistency**: Unified design language across all components

## Glassmorphic Effects

### Blur Levels
- **xs**: 2px - Subtle glass effect
- **sm**: 4px - Light glass effect
- **md**: 12px - Standard glass effect (default)
- **lg**: 20px - Strong glass effect
- **xl**: 40px - Maximum glass effect

### Glass Opacity
- Light mode: `rgba(255, 255, 255, 0.1)`
- Dark mode: `rgba(0, 0, 0, 0.1)`

### Glass Borders
- Light mode: `rgba(255, 255, 255, 0.2)`
- Dark mode: `rgba(255, 255, 255, 0.1)`

## Responsive Breakpoints

| Breakpoint | Size | Usage |
|------------|------|-------|
| xs | 320px | Small mobile devices |
| sm | 640px | Large mobile devices |
| md | 768px | Tablets |
| lg | 1024px | Desktop |
| xl | 1280px | Large desktop |
| 2xl | 1440px | Wide screens |

## Dark Mode

The design system includes full dark mode support with:
- Automatic system preference detection
- Manual theme toggle
- Persistent theme selection
- Smooth transitions between themes

## Accessibility

All components are designed with accessibility in mind:
- Proper ARIA labels
- Keyboard navigation support
- Focus indicators
- Color contrast ratios meeting WCAG 2.1 AA standards
- Screen reader compatibility

## Performance Considerations

### Mobile Optimization
- Backdrop filters are performance-optimized
- Blur effects use GPU acceleration
- Transitions are hardware-accelerated
- Critical CSS is inlined for faster initial render

### Browser Support
- Chrome 76+
- Firefox 103+
- Safari 9+
- Edge 79+

Note: Older browsers will gracefully degrade to solid backgrounds without blur effects.
